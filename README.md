Інструкція користувача 

1) Підготовка
1.	Переконайтесь, що встановлено JDK (Java Development Kit).
2.	Відкрийте термінал у папці з проєктом (або запускайте з IntelliJ).
3.	Якщо запускаєте з командного рядка — скомпілюйте програму:
      javac Main.java
      (У IntelliJ компіляція виконується автоматично.)
      Важливо: файл LOTR.txt (або ваш текстовий файл) має бути доступний за шляхом, який ви передаєте в аргументах. Якщо передаєте просто LOTR.txt, він повинен лежати у working directory запуску.

2) Архівація (стиснення)
   Формат запуску:
   java Main -k -N <path_to_text_file>
   Приклад (N = 2):
   java Main -k -2 LOTR.txt
   Результат:
   •	створиться архівний файл LOTR.txt.huf (поруч із LOTR.txt),
   •	у консолі буде виведено час стискання та розміри до/після.
________________________________________
3) Розархівація (відновлення)
   Формат запуску:
   java Main -d -N <path_to_archive_file>
   Приклад:
   java Main -d -2 LOTR.txt.huf
   Результат:
   •	буде створено відновлений файл поруч із архівом:
   o	якщо LOTR.txt ще не існує — створиться LOTR.txt,
   o	якщо існує — створиться decoded_LOTR.txt,
   •	у консолі буде виведено час розархівації.
4) Тест
   Запуск:
   java Main --test
   Поведінка:
   •	якщо у робочій папці є LOTR.txt — програма стискає та розтискає саме його і перевіряє коректність;
   •	якщо LOTR.txt немає — використовується внутрішній тестовий приклад.
